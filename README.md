# Task 3 - To-Do List CLI Application

## 📌 Objective
Create a command-line To-Do List application using Python that allows users to manage their daily tasks.

## 🚀 Features
- Add a new task
- View all tasks
- Delete a task
- Exit

## 🛠️ Technologies Used
- Python 3
- File Handling
- Functions
- Lists
- Loops
- Conditional Statements

## 📂 Project Structure
```
Task3-TodoList/
│── main.py
│── tasks.json
│── README.md
```

## 📸 Sample Output

```

===== TO-DO LIST =====
1. Add Task
2. View Tasks
3. Delete Task
4. Exit
Enter your choice (1-4): 
```
